# BULL-FARM

Tema:
O tema do projeto seria a criação de um programa em C++ onde o fazendeiro pode fazer o cadastro de seu gado, colocando
informações e características do animal, após o cadastro o fazendeiro pode identificar o
animal no programa digitando o código de sua marcação assim mostrando suas
informações e uma foto da raça do animal.
